## Getting Started

Install python requirements (use venv if needed):

```zsh
python3 -m venv ~/.venv
source ~/.venv/bin/activate
pip3 install -r requirements.txt
```

## Creating Plots for Times

```zsh
python3 average_times.py
python3 plot.py
```

## Creating Tables for Metrics

```zsh
python3 average_metrics.py
python3 average_metrics.py --synthetic
```
